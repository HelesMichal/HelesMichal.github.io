RefTrack

Basketball Referee Gesture Recognition System

A standalone AI device that watches a basketball referee, recognizes their hand signals in real time, and logs a human-readable match record for the scorer's desk — built end-to-end on a Raspberry Pi 5, from a self-recorded dataset to a final, boxed-up product.

The problem

Scorer's-desk staff have to track every referee call — fouls, jump balls, 24-second violations, substitutions — by eye, in real time, while also running the game clock and score. RefTrack takes part of that responsibility off their plate: a camera watches the referee, an on-device model classifies the signal, and the call is logged automatically so the desk can stay focused and make fewer mistakes during fast-paced stretches of the game.

How it's built

Custom vision model

Trained a YOLO model on a self-recorded dataset of referee signals — Ball Left/Right, Foul, 24-second violation, Jumpball, Substitution — captured across different people, environments, and lighting, then augmented for robustness.

Full-stack app

FastAPI backend handles all database CRUD and hardware I/O; Gradio powers a multi-tab dashboard (about, live data, operating, debugging) for testing and for the final product UI.

Dockerized data layer

PostgreSQL runs in Docker via compose, with six tables (Games, Records, ActionType, Teams, Locations, AuthenticationUsers) tying an RFID-authenticated scan to a game session and every subsequent AI detection.

Real hardware

RFID authentication gates access, a GPIO LED confirms boot and each new record, a USB camera streams to the live OpenCV/YOLO pipeline, and a physical button starts/stops the system alongside the touchscreen UI.

Deployed, not demoed

Runs as systemd services with auto-boot on the Raspberry Pi — a fully standalone device, not a laptop demo, housed in a laser-cut plywood enclosure with a foldable 7-inch touchscreen.

On-device inference

The YOLO model runs directly on the Pi against live camera footage; results flow to FastAPI, which updates the database and triggers hardware feedback in real time — no cloud round-trip.
